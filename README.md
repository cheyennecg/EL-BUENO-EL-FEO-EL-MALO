# LA POKEDEX DE LA VISUALIZACIÓN

# 🗺️ Kanto Battle Map:

¡Bienvenido a mi primer proyecto de mapa interactivo de la región de Kanto! Este mapa permite explorar los gimnasios y conocer a sus líderes. A través, de este juego, mi objetivo es aprender a usar filtros, tablas  y crear buenas gráficas. 
Las pistas de estas últimas las encontrarás en otro la carpeta VISUALIZACION - EL BUENO - EL FEO-EL MALO.

## 🔍 Guía del Entrenador (Pistas)
* **Ciudad Plateada:** El líder tiene una voluntad de roca. ¿Podrás encontrar su punto débil?
* **Pueblo Lavanda:** Algo extraño sucede aquí... el mapa se siente diferente en esta zona. ¿Podrás con ello?
* **Ciudad Celeste:** No te dejes engañar por la belleza del agua, el "Líder" de esta ciudad brilla con luz propia.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura del mapa y los nodos de los gimnasios.
* **CSS3:** Diseño retro, rutas (caminos) y posicionamiento absoluto.
* **JavaScript:** Lógica para actualizar el panel de batalla de forma dinámica.
* **GoogleSheets:** Para aprender a hacer tablas dinámicas y una buenas gráficas.

## 🚀 Cómo usar el mapa
1. Abre el enlace del proyecto.
2. Haz clic en los **puntos rojos** para desplegar la información del líder y sus pistas para saber cuales serán los top 3 pokemos con los que combatir en cada ciudad.
3. Consulta el panel derecho para ver quién es el Pokémon estrella (**Líder**) de cada gimnasio.
